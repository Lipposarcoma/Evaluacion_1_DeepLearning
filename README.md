## Instrucciones para ejecutar el proyecto
Este proyecto fue desarrollado en Google Colab para evitar problemas de dependencias locales. Para revisarlo y ejecutarlo, sigue estos pasos:

1. Haz clic en el botón azul **"Open in Colab"** que se encuentra en la parte superior del archivo `.ipynb` dentro de este repositorio.
2. Una vez que el cuaderno se abra en tu navegador, ve al menú superior y selecciona **Entorno de ejecución** (Runtime).
3. Haz clic en **Ejecutar todo** (Run all).
4. El cuaderno se ejecutará de manera secuencial: descargará automáticamente el dataset de Keras, aplicará la normalización, entrenará el modelo base y finalmente la versión optimizada con Dropout, mostrando todos los gráficos de rendimiento.
